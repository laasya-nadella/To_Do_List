# To-Do List Application (Python)

Developed a console-based task manager application using **Python** with a focus on clean, modular code.

- **File I/O Persistence:** Tasks are saved locally, so data remains between sessions.
- **CRUD Operations:** Add, view, update, and delete tasks efficiently.
- **OOP Principles:** Structured using classes for easy maintainability and scalability.

**Technologies Used:** Python, File I/O, Object-Oriented Programming

**Repository:** [GitHub](https://github.com/laasya-nadella/To_Do_List)
